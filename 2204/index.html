WordPress Hosting 性能指标

这里推荐（备忘）一个非常重要的测试一众知名WordPress托管商服务性能的网站：WordPress Hosting Performance Benchmarks

这个评测网站通过LoadStorm、K6 (Formerly Load Impact),WebPageTest.org,Uptime Monitoring,WPPerformanceTester,Qualsys SSL Report Grade,Internet.nl,Mozilla Observatory等知名工具进行了客观的测试（详见测试方法），然后综合各指标、按照不同的价格区间进行综合排名，并得出推荐星级。

他们会每年对这些服务商开展一次横评，并且，会把历年的数据拿来对照，通过横向对比找差距、历史对比看进步，来对服务质量进行全面的了解，让我们消费者能更好的进行决策。

例如，EasyWP，就是目前我使用托管商，在该网站的的测试数据，还算不错，而2021年的成绩，比2020年还有一定进步，在便宜的服务商里面，获得了好的评价（Top Tier）。（查看详细测试）
