Adguard Home 53端口被占用

Docker安装Adguard Home，提示53端口被占用，这种情况下，Adguard Home 的核心功能DNS解析就不起作用，这样肯定不行。

1.安装netstat

sudo apt update && sudo apt install net-tools #ubuntu

netstat -tulpn

可以看到，systemd-resolve这个ubuntu用于DNS解析的进程占用了53端口。

2.结束systemd-resolved

sudo systemctl disable systemd-resolved.service sudo systemctl stop systemd-resolved

再看，53端口已经释放

3.重新部署Adguard Home，就可以运行了。
